# Find Your Karas

This application identifies fish species, specifically differentiating between **Karas obecný** and **Karas stříbřitý**.

## Setup

1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python app.py
   ```

## Training

Refer to the `Training` folder for instructions on how to train the model using your Roboflow dataset.
